§ 1 — Allowable Stress Formulae (API-620 Cl. 3.3.2 & 6.2.2)

Normal Operating Allowable Stress (Sd)
API-620 Clause 3.3.2 | Weld Metal
Sd = min( 2·Fy / 3 , UTS / 3 )
Fy = Yield strength of weld metal (MPa); UTS = Tensile strength of weld metal (MPa)
Hydrotest Allowable Stress (Sht)
API-620 Clause 6.2.2 | Weld Metal
Sht = min( 0.85·Fy , 0.55·UTS )
Applied during hydrostatic test condition only
Seismic Allowable Stress (OLE / CLE)
API-620 Appendix Q
S_seis = min( 1.33·Sd , 0.8·Fy )
Used for both OLE and CLE load combinations
Internal Pressure Design Stress
API-620 Cl. 3.3.1
Sd (base) = min( 2·Fy_base / 3 , UTS_base / 3 )
Applies to base metal for pressure loading

§ 2 — Shell Thickness Formulae (API-620 Cl. 5.3)

Design Shell Thickness (td)
API-620 Clause 5.3.2 — Hydrostatic + Pressure
td = [ 4.9 · D · (Hd + Pi) · G ] / (Sd · E) + CA
D = Inside dia. (m); Hd = Liquid head height (m); Pi = Internal pressure (m barg); G = Specific gravity; E = Joint efficiency; CA = Corrosion allowance (mm)
Hydrotest Shell Thickness (tt)
API-620 Clause 6.2 — Hydrotest Condition
tt = [ 4.9 · D · (Ht + Pt) · ρw×10⁻³ ] / (Sht · E)
Ht = Hydrotest liquid level (m); Pt = Test pressure (m barg); ρw = Water density (kg/m³)
Required Shell Thickness
Governing condition
t_req = max( td , tt , t_min )
t_min = Code minimum thickness (mm); adopted plate thickness must be ≥ t_req
Course Weight
Geometry calculation
W_course = π · D · t_adopted · w · ρ_steel / 1000
w = Course width (m); ρ_steel = Steel density (kg/m³). Result in kg.

§ 3 — Seismic Natural Period (API-620 Appendix Q, Section Q.3)

Impulsive Natural Period (Ti)
API-620 Appendix Q, Eq. Q-4
Ti = Ci · Hd · √(ρ · D) / √(2000 · tU · E)
Ci = Dimensionless period coefficient (6–9.5); Hd = Design liquid level (m); ρ = Liquid density (kg/m³); tU = Equivalent uniform shell thickness (mm); E = Elastic modulus (MPa)
Convective Natural Period (Tc)
API-620 Appendix Q, Eq. Q-5
Tc = (0.578 / √[tanh(3.68·Hd/D)]) × 1.8 × √D
D = Tank inner diameter (m); Hd = Design liquid level (m). Tc typically 5–15 s for large tanks.

§ 4 — Effective Liquid Masses (API-620 Appendix Q, Section Q.3.2)

Impulsive Effective Weight (Wi) — Tall Tank D/H ≥ 1.333
API-620 Appendix Q, Eq. Q-8a
Wi = [ tanh(0.866·D/Hd) / (0.866·D/Hd) ] · Wp
Wp = Total weight of liquid contents (N)
Impulsive Effective Weight (Wi) — Squat Tank D/H < 1.333
API-620 Appendix Q, Eq. Q-8b
Wi = (1 − 0.218·D/Hd) · Wp
Height of Wi from base: Xia = (0.5 − 0.094·D/Hd)·Hd
Impulsive C.O.G. Height (Xia) — Tall Tank
API-620 Appendix Q, Eq. Q-10a
Xia = 0.375 · Hd
Applied when D/H ≥ 1.333
Convective Effective Weight (Wc)
API-620 Appendix Q, Eq. Q-9
Wc = 0.23·(D/Hd)·tanh(3.67·Hd/D) · Wp
Height of Wc from base: Xc = [1 − (cosh(3.67·Hd/D)−1) / ((3.67·Hd/D)·sinh(3.67·Hd/D))] · Hd

§ 5 — Seismic Forces & Overturning Moment (API-620 Appendix Q, Q.3.3)

Total Seismic Base Shear (V)
API-620 Appendix Q, Eq. Q-14
V = √[ (Ai·(Ws+Wi+Wr+Wf))² + (Ac·Wc)² ]
Ai = Impulsive spectral acceleration; Ac = Convective spectral acceleration; Ws = Shell weight; Wr = Roof weight; Wf = Bottom plate weight (all in N)
Ringwall Overturning Moment (Mrw)
API-620 Appendix Q, Eq. Q-15
Mrw = √[ (Ai·(Wi·Xia+Ws·Xs+Wr·Xr))² + (Ac·Wc·Xc)² ]
Xs = Shell C.O.G. height; Xr = Roof C.O.G. height; Xia = Impulsive liquid C.O.G.; Xc = Convective liquid C.O.G. (all in m)

§ 6 — Anchorage & Shell Compression (API-620 Appendix Q, Q.3.4)

Shell Dead-Weight Force per Unit Length (wt)
API-620 Appendix Q
wt = Ws / (π·D) + wrs
wrs = Roof load distributed onto shell (N/m)
Resisting Uplift Force (wa)
API-620 Appendix Q, Eq. Q-18
wa = min[ 99·ta·√(FyA·ρ/1000·(1−0.4Av)·Hd) , 201.1·Hd·D·ρ/1000·(1−0.4Av) ]
ta = Annular plate thickness (mm); FyA = Annular plate yield strength (MPa); Av = Vertical seismic coefficient
Anchorage Ratio (J)
API-620 Appendix Q, Eq. Q-17
J = Mrw / [ (wt·(1−0.4Av) + wa − 0.4·wint) · D² ]
J ≤ 0.785 → Self-anchored (no uplift); J > 1.54 → Mechanically anchored required
Actual Shell Compression (σc)
API-620 Appendix Q, Eq. Q-19 / Q-20
σc = (wt·(1+0.4Av) + 1.273·Mrw/D²) / (1000·ts) [Anchored] σc = ((wt·(1+0.4Av)+wa)/(0.607−0.18667·J^2.3)−wa) / (1000·ts) [J>0.785]
ts = Bottom shell course thickness (corroded, mm); Must satisfy σc < Fc
Allowable Shell Compression (Fc) — A ≥ 44
API-620 Appendix Q, Eq. Q-21a
Fc = min( 0.5·Fy , 83·ts/D ) [when A ≥ 44] A = ρ/1000 · Hd · D² / ts²
A = Geometric-loading parameter; Fy = Bottom shell yield strength (MPa)
Allowable Shell Compression (Fc) — A < 44
API-620 Appendix Q, Eq. Q-21b
Fc = min( 0.5·Fy , 83·ts/(2.5·D) + 7.5·√(G·H) ) [A<44]
G = Specific gravity of liquid; H = Design liquid level (m)

§ 7 — Dynamic Hoop Force Formulae (API-620 Appendix Q, Q.3.5)

Hydrostatic Hoop Force (Ps)
API-620 Appendix Q
Ps = ρ × 4.9 × Y × D [N/m]
Y = Height from bottom to course level (m); D = Inner diameter (m); ρ = Liquid density (kg/m³ × 10⁻³)
Impulsive Hoop Pressure (Pi) — D/H ≥ 1.333
API-620 Appendix Q, Eq. Q-23a
Pi = 8.48×10³·Ai·G·D·Hd·[(Y/Hd)−½·(Y/Hd)²]·tanh(0.866·D/Hd)
Ai = Impulsive spectral coefficient; G = Specific gravity
Impulsive Hoop Pressure (Pi) — Y < 0.75D
API-620 Appendix Q, Eq. Q-23b
Pi = 5.22×10³·Ai·G·D²·[(Y/0.75D)−½·(Y/0.75D)²]
Convective Hoop Pressure (Pc)
API-620 Appendix Q, Eq. Q-24
Pc = 1.85·Ac·G·D²·cosh(3.68·(Hd−Y)/D) / cosh(3.68·Hd/D) × 10³
Ac = Convective spectral coefficient
Vertical Seismic Hoop Pressure (Pv)
API-620 Appendix Q
Pv = Ps × Av
Av = Vertical seismic design coefficient
Combined Pressure by RSS Method (PRSS)
API-620 Appendix Q, Eq. Q-25
PRSS = Ps + √(Pi² + Pc² + Pv²) [N/m]
Required thickness check: TRSS = PRSS / S_allow; must be ≤ t_adopted
Max Horizontal Hoop Force (Ph)
API-620 Appendix Q, Eq. Q-26
Ph_max = (Ps + Pi + Pc + 0.4·Pv) × 10⁻³ [N/mm]
Th = Ph_max / S_allow (required thickness by horizontal check)
Max Vertical Hoop Force (Pv_max)
API-620 Appendix Q, Eq. Q-27
Pv_max = (Ps + Pv + 0.3·(Pi + Pc)) × 10⁻³ [N/mm]
Tv = Pv_max / S_allow (required thickness by vertical check)

§ 7 — External Pressure (Perlite Insulation)

Friction Coefficient
f = tan(0.75 · θ)
θ = response angle of perlite (°). Lower θ gives higher, more conservative pressure.
Horizontal Pressure Ratio
Rankine earth pressure
k = max((1−sinθ)/(1+sinθ), 0.3)
Ratio of horizontal to vertical pressure in the perlite annulus.
Thermal Shrinkage Displacement
Excel J17
dT = α · (Ta − TL) · R
Radial shrinkage of inner tank on cool-down. α = linear expansion coeff., R = radius.
Perlite Layer Thickness
Excel J15 / J16
L = (L₁ + dT) − δg + δg·γ/100
Effective thickness filled with perlite. δg = glass wool blanket thickness; γ = compressed ratio (%).
Perlite Pressure (Cool-down)
Excel J19–J20
Pp = (L·Wp / 2f) · (1 − e^(−2kfh/L))
Horizontal pressure on outer shell from perlite powder. Wp = density; h = insulation height.
Design External Pressures
Table 5
P_cool = Pp · γF,cool  |  P_warm = Pp,wu · γF,warm
γF,cool = 1.36 (operating); γF,warm = 1.06 (testing). Warm-up uses Pp×2.4 for glass-wool compression.

§ 8 — Shell Stiffener Design Formulae

Equivalent Shell Height (H_eq)
/ EN 1993-4.2
H_eq = Σ [ W_i · (t_min / t_i)^2.5 ]
W_i = course width (m); t_i = adopted course thickness (mm); t_min = minimum shell thickness (mm). Converts variable thickness shell to uniform equivalent.
Shell Buckling Check Requirement
§9 / D.1
Buckling check required if: r/t > 0.21·√(E/fyk)
r = shell radius (mm); t = shell thickness (mm); E = Young's modulus (MPa); fyk = yield strength (MPa)
Omega – Cylinder Length Parameter
D.1.2
Ω = L_s / √(r · t)
L_s = stiffener spacing (mm). Used to classify cylinder as Short (Ω/C_θ < 20), Medium, or Long (Ω/C_θ > 1.63·r/t)
C_theta – Boundary Condition Factor
Table D.4 (Short Cylinders)
BC1-BC1: 1.5 + 10/Ω² − 5/Ω³ BC1-BC2: 1.25 + 8/Ω² − 4/Ω³ BC2-BC2: 1.0 + 3/Ω^1.35 BC1-BC3: 0.6 + 18/Ω² − 0.3/Ω³
BC1 = Clamped, BC2 = Pinned, BC3 = Free. Medium/Long cylinders use fixed C_θ values.
Critical Buckling Stress (σ_Rcr)
D.1.2.2
σ_Rcr = 0.92 · E · t / (r · (Ω/C_θ))
Elastic critical circumferential buckling stress under external pressure
Slenderness & Reduction Factor
§8.5 / D.1.3
λ_θ = √(fyk / σ_Rcr) λ_θp = √(α_θ / (1 − β)) χ_θ = α_θ / λ_θ²
α_θ = imperfection factor (from fabrication quality class); β = plastic range factor; γ_M1 = material partial factor
Design Buckling Resistance
§8.5.2
σ_theta,Rd = χ_θ · fyk / γ_M1
Must satisfy: σ_theta,Ed < σ_theta,Rd; where σ_theta,Ed = Pe · r / t
API-620 Maximum Stiffener Spacing
API-620 Section 5.10.6
L_max = 600·t_in·(100·t_in / (D_ft))^1.5 [converted to mm]
t_in = thickness in inches; D_ft = diameter in feet. Alternate empirical check.
Number of Stiffeners Required
Derived from equivalent height
